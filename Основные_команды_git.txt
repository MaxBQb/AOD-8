1) git pull                 подгружает все новые данные с сервера
2) git status               узнать ветку
3) git branch {имя ветки}   сменяет текущую ветку
4) git log                  все текущие коммиты
5) git log --oneline        показ все коммитов в более удобном режиме
6) пока все, если еще что-то нужным посчитаю, допишу
